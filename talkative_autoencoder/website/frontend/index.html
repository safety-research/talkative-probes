What is this? (click to expand)

Natural language autoencoders (or talkative autoencoders) are a research tool for interpreting LLMs by:

  • Converting the model's internal neural activations into natural language explanations
  • Testing if these explanations can accurately reconstruct the original activations
  • Hopefully showing what the model "thinking" at each token in the text

Lower MSE values indicate the explanation better captures what the model is doing internally.

Understanding the Results

Position: Token index in the input sequence

Token: The actual text token being analyzed

Explanation: Human-readable interpretation of what the model is "thinking" at this token

MSE: Mean Squared Error - measures how well the explanation reconstructs the model's internal state (lower is better)

Relative RMSE: The RMSE divided by the norm of the original vector.

Salience: Importance of this token - % increase in MSE when replaced with a space token (higher = more important)
